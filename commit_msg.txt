feat: Implementa melhorias abrangentes no chatbot

Esta atualização robustece o chatbot 'JavisGB' com várias funcionalidades novas, melhorias de performance e correções críticas.

Principais Melhorias:
- Respostas em Tempo Real (Streaming): A interface agora exibe as respostas do bot palavra por palavra, melhorando significativamente a experiência do usuário.
- Exibição de Fontes: Cada resposta agora inclui uma lista dos documentos de origem utilizados pela IA, aumentando a transparência e a confiança do analista.
- Coleta de Feedback: Foram adicionados botões de feedback (👍/👎) a cada resposta, salvando os dados em um arquivo 'feedback.csv' para permitir a análise e melhoria contínua do sistema.
- Extração de Metadados Aprimorada: O script de indexação foi refatorado para extrair metadados (título, código) diretamente do conteúdo dos PDFs, tornando o processo mais robusto.

Correções e Refatorações:
- Resolução de Conflitos: Corrigidos conflitos de merge do Git nos arquivos 'chatbot.py' e 'app.py'.
- Modernização da Cadeia de IA: A lógica do chatbot foi refatorada para usar a sintaxe moderna LCEL (LangChain Expression Language), tornando o código mais limpo e flexível.
- Correção de Ambiente Virtual: O ambiente 'venv' foi recriado para corrigir problemas de execução e garantir a consistência das dependências.
- Aprimoramento do .gitignore: O arquivo '.gitignore' foi atualizado para ignorar corretamente pastas 'node_modules' e outros arquivos gerados.
